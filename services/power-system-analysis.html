Power System Analysis

Sound engineering decisions start with an accurate model. NERX conducts power system studies across multiple software platforms — CAPE, ETAP, SKM, and others — so we can work in the toolset your organization already owns instead of forcing a conversion.

Beyond individual studies, we serve as custodians of client system models: keeping them current as the system changes, so the next study starts from a trustworthy baseline rather than a modeling project.

Typical Scope & Deliverables

  • Short circuit studies
  • Protective device coordination studies
  • Power flow and voltage studies
  • Motor starting analysis
  • Stability analysis
  • System model development and custodianship
