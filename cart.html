How this order works

  • The purchaser is whoever pays at checkout — the PayPal account (or debit/credit card) used. The receipt and the full list of ticket codes go to that email address. If a PayPal account payment has issues, use the debit or credit card option instead.
  • Two emails to the purchaser: a receipt for the full order (every training, price, and the PayPal transaction number), and a separate ticket codes email listing every code and who each one is assigned to.
  • Seats marked for someone else are registered to that person: their name appears on the class roster and their ticket code is emailed to them directly.
  • Seats reserve automatically. Each seat books onto its session the moment payment completes. If a session fills up first, that code still arrives — unassigned — and can be placed on any other date.
  • Rescheduling: any ticket can be moved to a different session at the calendar page up to one hour before its session starts.
  • Questions, changes, or refunds: contact Doug Millner at douglas.millner@nerxpower.com. A PayPal refund automatically cancels the order's ticket codes.
